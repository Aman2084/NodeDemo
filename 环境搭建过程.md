============== 1. 安装 ==============
npm init -y
npm install -g typescript
npm install -g ts-node
npm i @types/node -D

============== 2. 配置 ==============
    npm config get prefix       // 获取地址

    桌⾯“此电脑”右键点击“属性”
    窗⼝左侧列表点击“⾼级系统设置”
    新窗⼝点击下⽅“环境变量”
    “系统变量”选择“Path”，点击“编辑”
    点击“新建”，将 运行命令  中得到的路径复制进去，点击保存
    重启计算机
